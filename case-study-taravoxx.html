The Problem

Every Astrology Page Looks the Same

The niche was crowded with loud, indistinguishable content — making it impossible to build trust or stand out.

  • Overused gold palettes and heavy typography
  • Random stock images — temples, gods, planets
  • No consistent brand identity across posts
  • Visually cluttered feeds with zero recall
SCREENSHOT — Typical Astrology Reels Feed
Strategy

Go Minimal Where Others Go Loud

The brief was simple: build what no astrology account had — a calm, modern, educational brand voice.

  • Calm visual language over high-energy clutter
  • Modern minimal editing — let the content breathe
  • Clear information hierarchy for fast comprehension
  • Consistent brand identity built to scale
SCREENSHOT — Taravoxx Feed Grid
Visual Identity

A System Built for Recall

Every frame follows the same visual rules — so the brand is recognisable even mid-scroll.

Purple + Beige Palette Clean Modern Typography UI-style Info Cards AI Illustration Visuals Subtle Transitions
SCREENSHOT — Brand Color & Typography Reference
Editing Style

Premium Feels Calm, Not Flashy

The editing was designed to feel trustworthy — like a knowledgeable friend, not a hype reel.

  • Minimal transitions — blur and fade only
  • Ambient background music, no aggressive SFX
  • Creator's face always front and center
  • Subtle noise and tint for cinematic texture
SAMPLE REEL THUMBNAILS — Spiritual / Kalsarpa / Rudraksha
Visual Storytelling

Replace Stock with Story

AI-generated illustrations replaced generic stock imagery — making complex astrology concepts instantly visual.

  • Planetary deities rendered with consistency and detail
  • Symbolic visuals aligned with each reel topic
  • Beige background cards for clean in-feed contrast

Content System

Three Formats. Infinite Reels.

Content was structured into repeatable formats so the team could produce consistently without reinventing the wheel.

01

Educational

Talking-head explanations of astrology concepts. Retention-paced, hook-driven.

02

Concept Visual

Illustration-based reels. Planetary ideas visualised for discovery.

03

Story Reel

Symbolic visuals designed to attract reach and new audiences.

Scalable System

Built to Grow Without Breaking

A reusable editing template was built so the brand stays consistent — regardless of volume or team size.

  • Reusable Premiere Pro project files
  • Typography rules locked across all formats
  • Pre-built UI components and visual presets
  • Plug-and-play for any new editor onboarded
